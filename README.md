# 🌤️ React Weather App

A simple and responsive weather application built with React.  
It allows users to search for any city and get real-time weather information using a weather API.

---

## 🚀 Features

- 🌡️ Display current temperature  
- 🌥️ Show weather conditions (clear, cloudy, rain, etc.)  
- 📍 Real-time API data fetching  
- 📱 Responsive UI for mobile and desktop  

---

## 🛠️ Tech Stack

- React.js (Hooks: useState, useEffect)
- JavaScript (ES6+)
- CSS3 / Material UI (depending on your project)
- Weather API (open meteo api)
- Axios to fetch api
